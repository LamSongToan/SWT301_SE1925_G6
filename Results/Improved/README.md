# Improved Results README / Tài liệu kết quả Improved

## 1. Overview / Tổng quan

English:

This folder stores output files generated from LLM experiments on the Improved
bug report dataset. The Improved dataset contains bug reports after quality
improvement, especially in Steps to Reproduce, Observed Behavior, and Expected
Behavior.

Tiếng Việt:

Thư mục này lưu các file kết quả được tạo ra từ thí nghiệm LLM trên bộ dữ liệu
Improved. Bộ dữ liệu Improved gồm các bug report đã được cải thiện chất lượng,
đặc biệt ở phần Steps to Reproduce, Observed Behavior, và Expected Behavior.

## 2. Current Status / Trạng thái hiện tại

English:

At the current stage, this folder is only prepared for future outputs. Empty
result files should not be committed because they may be mistaken for completed
experiment outputs.

Tiếng Việt:

Ở giai đoạn hiện tại, thư mục này chỉ được chuẩn bị để chứa kết quả sau này.
Không nên commit các file kết quả rỗng vì dễ gây hiểu nhầm rằng thí nghiệm đã
được chạy xong.

## 3. Expected Output Files / Các file kết quả dự kiến

The following files will be generated after running Improved-side experiments:

pilot_llm_output_improved.csv
pilot_api_log_improved.txt
full_llm_output_improved.csv
full_api_log_improved.txt
summary_improved.csv
mismatch_analysis_improved.csv

File descriptions:

- `pilot_llm_output_improved.csv`
  - English: LLM predictions for the Improved pilot dataset.
  - Tiếng Việt: Kết quả dự đoán của LLM trên tập Improved pilot.

- `pilot_api_log_improved.txt`
  - English: API call log for the Improved pilot experiment.
  - Tiếng Việt: Log gọi API cho thí nghiệm Improved pilot.

- `full_llm_output_improved.csv`
  - English: LLM predictions for the full Improved dataset.
  - Tiếng Việt: Kết quả dự đoán của LLM trên toàn bộ tập Improved.

- `full_api_log_improved.txt`
  - English: API call log for the full Improved experiment.
  - Tiếng Việt: Log gọi API cho thí nghiệm Improved full.

- `summary_improved.csv`
  - English: Metric summary for Improved results, including accuracy and Cohen's Kappa.
  - Tiếng Việt: Tổng hợp metric cho kết quả Improved, gồm accuracy và Cohen's Kappa.

- `mismatch_analysis_improved.csv`
  - English: Cases where LLM prediction differs from Improved ground truth.
  - Tiếng Việt: Các case mà dự đoán của LLM khác với ground truth Improved.

## 5. Data Rules / Quy tắc dữ liệu

English:

- Do not manually create empty result CSV or TXT files.
- Result files should be generated by experiment scripts.
- Keep Improved outputs separate from Raw outputs.
- Store Raw-side outputs in `Results/Raw/`.

Tiếng Việt:

- Không tự tạo file CSV hoặc TXT kết quả rỗng.
- Các file kết quả nên được sinh ra bởi script thí nghiệm.
- Giữ output Improved tách riêng với output Raw.
- Output của Raw cần được lưu trong `Results/Raw/`.
