# Raw Results README / Tài liệu kết quả Raw

## 1. Overview / Tổng quan

English:

This folder stores output files generated from LLM experiments on the Raw bug
report dataset. The Raw dataset is the original version of the bug reports
before improvement.

Tiếng Việt:

Thư mục này lưu các file kết quả được tạo ra từ thí nghiệm LLM trên bộ dữ liệu
Raw. Bộ dữ liệu Raw là phiên bản bug report gốc trước khi được cải thiện.

## 2. Current Status / Trạng thái hiện tại

English:

At the current stage, this folder is only prepared for future outputs. Empty
result files should not be committed because they may be mistaken for completed
experiment outputs.

Tiếng Việt:

Ở giai đoạn hiện tại, thư mục này chỉ được chuẩn bị để chứa kết quả sau này.
Không nên commit các file kết quả rỗng vì dễ gây hiểu nhầm rằng thí nghiệm đã
được chạy xong.

## 3. Expected Output Files / Các file kết quả dự kiến

The following files will be generated after running Raw-side experiments:

pilot_llm_output_raw.csv
pilot_api_log_raw.txt
full_llm_output_raw.csv
full_api_log_raw.txt
summary_raw.csv
mismatch_analysis_raw.csv

File descriptions:

- `pilot_llm_output_raw.csv`
  - English: LLM predictions for the Raw pilot dataset.
  - Tiếng Việt: Kết quả dự đoán của LLM trên tập Raw pilot.

- `pilot_api_log_raw.txt`
  - English: API call log for the Raw pilot experiment.
  - Tiếng Việt: Log gọi API cho thí nghiệm Raw pilot.

- `full_llm_output_raw.csv`
  - English: LLM predictions for the full Raw dataset.
  - Tiếng Việt: Kết quả dự đoán của LLM trên toàn bộ tập Raw.

- `full_api_log_raw.txt`
  - English: API call log for the full Raw experiment.
  - Tiếng Việt: Log gọi API cho thí nghiệm Raw full.

- `summary_raw.csv`
  - English: Metric summary for Raw results, including accuracy and Cohen's Kappa.
  - Tiếng Việt: Tổng hợp metric cho kết quả Raw, gồm accuracy và Cohen's Kappa.

- `mismatch_analysis_raw.csv`
  - English: Cases where LLM prediction differs from Raw ground truth.
  - Tiếng Việt: Các case mà dự đoán của LLM khác với ground truth Raw.

## 5. Data Rules / Quy tắc dữ liệu

English:

- Do not manually create empty result CSV or TXT files.
- Result files should be generated by experiment scripts.
- Keep Raw outputs separate from Improved outputs.
- Store Improved-side outputs in `Results/Improved/`.

Tiếng Việt:

- Không tự tạo file CSV hoặc TXT kết quả rỗng.
- Các file kết quả nên được sinh ra bởi script thí nghiệm.
- Giữ output Raw tách riêng với output Improved.
- Output của Improved cần được lưu trong `Results/Improved/`.
